Lista de Presença




A lista de presença digital, feita em HTML, CSS e JavaScript, permite que registrem seus dados, com nome, numero de chamada, turma e e-mail. ao clicar em 'Incluir', as informações são salvas no banco de dados para monitorar a presença.

    Dados do formulário salvos temporariamente.
